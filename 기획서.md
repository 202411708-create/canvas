# ADHD 시간 관리 프로그램 기획서
## "나의 시간 사용 알아보기" (1회기)

---

## 1. 기획 의도

ADHD 학생들은 실행기능 장애로 인해 **시간 맹목성(Time Blindness)** 을 경험합니다. 자신이 시간을 어떻게 사용하는지 객관적으로 인식하지 못하며, 이는 학업 및 일상생활 관리의 어려움으로 이어집니다.

본 프로그램은 **게임화된 인터랙티브 방식**으로 자신의 시간 사용 패턴을 시각화하고, 시간 도둑(time thief)을 발견하여 **자기 인식(self-awareness)** 을 높이는 것을 목표로 합니다.

---

## 2. 대상

- **중고등학생 ADHD 학생** (중학교 1학년 ~ 고등학교 3학년)
- 시간 관리에 어려움을 겪는 학생
- 자신의 시간 사용 패턴을 파악하지 못하는 학생

---

## 3. 목적

### 주요 목적
- 자신의 24시간 시간 사용 패턴 파악
- 시간 낭비 요인(시간 도둑) 인식
- 시간 관리 개선 의지 형성

### 세부 목표
1. **시간 외재화**: 추상적인 시간을 구체적 시각 자료로 전환
2. **즉각적 피드백**: ADHD 특성에 맞는 실시간 반응 제공
3. **자기 조절 학습 1단계**: Zimmerman의 예비 사고 단계(forethought phase) 달성
4. **실행 의도 형성**: Gollwitzer의 If-Then 계획 수립 준비

---

## 4. 포함 요소

### 모듈 구성 (총 40분)

#### Module 0: 온보딩 (3분)
- 프로그램 소개 및 동기 부여
- 탐정 콘셉트 도입 (🔍 "시간 탐정이 되어보자!")
- 자동 진행 카운트다운

#### Module 1: 타임맵 - 어제 뭐 했지? (12분)
- **기능**: 어제 24시간 활동 기록
- **프로세스**:
  1. 수면 시간 입력 (기상/취침 시각)
  2. 시간대별 활동 선택 (등교 전/학교/방과후/저녁/밤)
  3. 활동별 소요 시간 입력
- **산출물**: 24시간 원형 그래프, 활동별 시간 통계
- **피드백**: 스크린 vs 공부 시간 비교, 수면 시간 분석

#### Module 2: 시간 도둑 카드 게임 (15분)
- **기능**: 10가지 시간 도둑 카드 중 자신에게 해당하는 것 선택
- **카드 종류**:
  - 📱 끝없는 숏폼 영상
  - 🎮 한 판만... 게임
  - 💬 카톡 확인하다가...
  - 📺 유튜브 자동 재생
  - 🛏️ 침대에 누워서 멍때리기
  - 🎧 음악 들으며 딴생각
  - 🍔 간식 먹으며 휴대폰
  - 🎨 꾸미기·정리에 집착
  - 💭 고민만 하고 시작 안 함
  - 🔄 완벽주의로 다시 하기
- **산출물**: 선택한 시간 도둑 목록 및 예상 손실 시간

#### Module 3: 다짐하기 (8분)
- **기능**: 시간 도둑 퇴치 방법 선택 및 실천 다짐
- **프로세스**:
  1. 시간 도둑별 구체적 대응 전략 선택
  2. If-Then 형식의 실행 의도 설정
  3. 자기 선언문 작성
- **산출물**: 개인 맞춤형 실천 계획서

#### Final Screen: 종료 화면
- 전체 결과 요약
- 격려 메시지
- 재시작 옵션

## 5. 이론적 근거

### 1) Barkley의 실행기능 이론
- **시간 외재화(Externalization of Time)**: ADHD는 내적 시간 표상 능력이 약하므로, 시각적 도구로 시간을 외부화

### 2) Zimmerman의 자기조절 학습 모델
- **예비 사고 단계(Forethought Phase)**: 목표 설정 및 전략적 계획 수립
- 본 프로그램은 1회기로서 이 단계에 집중

### 3) Gollwitzer의 실행 의도 이론
- **If-Then Planning**: "만약 ~라면, 나는 ~할 것이다" 형식의 구체적 계획
- 자동적 행동 실행을 촉진

### 4) 시간 지각 이론 (Time Perception Theory)
- ADHD의 시간 맹목성(Time Blindness) 보완
- 시각화를 통한 시간 인식 개선

### 5) 행동 수정 이론 (Behavior Modification)
- **즉각적 강화**: 0.3초 이내 피드백 제공
- **긍정적 강화**: 탐정 콘셉트, 격려 메시지

## 6. 주요 특징

### ADHD 친화적 UX/UI
- ✅ **즉각적 피드백** (<0.3초 반응 시간)
- ✅ **최소 작업 기억 부하** (한 번에 하나의 질문)
- ✅ **명확한 진행 표시** (1/3, 2/3, 3/3 진행바)
- ✅ **큰 터치 영역** (최소 44px 버튼)
- ✅ **애니메이션 피드백** (클릭 시 시각적 반응)
- ✅ **게임화 요소** (탐정 콘셉트, 카드 게임)

### 기술적 특징
- **반응형 디자인**: 모바일/태블릿/PC 모두 최적화
- **단일 페이지 애플리케이션(SPA)**: 페이지 전환 없는 부드러운 경험
- **경량화**: 외부 의존성 최소화 (lucide-react만 사용)
- **빠른 로딩**: Vite 기반 번들링
